## [0.0.9] - 2025-11-07

### Fixed

- Race condition that occurred when creating the default folder.

## [0.0.8] - 2025-11-05

### Added

- Drag and drop functionality to reorder commands and folders.

## [0.0.7] - 2023-11-03

### Added

- Initial release of the "Favorite CLI Commands" extension.
- View container in the activity bar with a custom icon.
- Tree view for displaying favorite commands, organized by folders.
- Default folder for commands.
- Commands to manage folders: Add, Rename, Delete.
- Commands to manage commands: Add, Edit, Delete.
- Commands to run or paste commands into the active terminal.
- Commands to import and export the command list to a JSON file.
- Configuration in `settings.json` to manage commands.
- Refresh button to reload the command list.
